Privacy Policy - D365 Translation Manager

Last Updated: December 23, 2024

Privacy Summary: D365 Translation Manager does not collect, store, or transmit any personal data or user information to external servers. All operations occur locally within your browser and communicate only with your Microsoft Dynamics 365 environment.

Data Collection

D365 Translation Manager does not collect, store, or transmit any of the following:

  • Personal information (name, email, phone number)
  • Authentication credentials or passwords
  • Browsing history or activity
  • Financial or payment information
  • Location data
  • Health information
  • Personal communications

Data Processing

All data processing occurs locally within your browser. The extension communicates only with your Microsoft Dynamics 365 environment using the standard Web API to:

  • Read entity and attribute metadata
  • Update entity and attribute labels
  • Read and parse form structures (XML)
  • Update form control labels
  • Read and update global option sets
  • Retrieve plugin trace logs

No data is sent to third-party servers or external services.

Required Permissions

The extension requires the following permissions to function:

scripting

Used to inject content scripts that highlight translatable fields on Dynamics 365 forms, helping users identify which fields require translation.

activeTab

Used to detect when you're on a valid Dynamics 365 page and capture context information (current entity, attribute, form) to open the appropriate editor.

tabs

Used to open report pages (Entity Browser, Form Editor, OptionSet Manager, Plugin Trace Logs) in new tabs for full-screen editing experience.

browsingData

Used to clear browser cache and perform hard refresh when requested through the Developer Tools tab. This ensures metadata changes are properly reloaded.

storage

Used to store user preferences such as theme selection (light/dark mode), interface layout settings, and display configurations. No sensitive data is stored.

host_permissions (https://*.dynamics.com/*)

Required to communicate with the Dynamics 365 Web API across all user environments. All communication uses your existing Dynamics 365 authentication and occurs over HTTPS.

Third-Party Services

This extension does not use any third-party services including:

  • Analytics or tracking services
  • Advertising networks
  • External APIs or servers
  • Social media integrations

Data Security

Security measures include:

  • All communication with Dynamics 365 uses your existing authentication
  • All API calls are made over HTTPS
  • No credentials are stored by the extension
  • Extension operates in isolated browser context
  • Chrome Manifest V3 compliance for enhanced security

Data Retention

The extension stores only non-sensitive user preferences (theme, layout) in Chrome's local storage. This data:

  • Remains on your device
  • Is never transmitted externally
  • Can be cleared by uninstalling the extension
  • Can be reset through browser settings

User Rights

You have the right to:

  • Review stored preferences in Chrome's extension storage
  • Clear all extension data by uninstalling
  • Request information about data processing
  • Submit questions or concerns about privacy

Children's Privacy

This extension is designed for business use with Microsoft Dynamics 365 and is not directed at children under 13 years of age. We do not knowingly collect information from children.

Updates will be posted on this page with a new "Last Updated" date. Continued use of the extension after changes constitutes acceptance of the updated policy.
